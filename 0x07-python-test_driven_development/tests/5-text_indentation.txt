Text Indentation
================

This module contains a function for printing a text with 2 new lines after each of these characters: '.', '?', and ':'.

Function:
---------

text_indentation(text)

Description:
------------

The `text_indentation` function prints the input text with 2 new lines after each occurrence of the characters '.', '?', and ':'.

Parameters:
-----------

- text (str): The text to be processed.

Returns:
--------

None

Raises:
-------

- TypeError: If the input text is not a string.

Example Usage:
--------------

```python
